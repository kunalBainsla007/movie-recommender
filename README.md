# Movie Recommender System

## What is this project?
A simple movie recommendation system built with Python that suggests similar movies based on genre similarity.

## Setup Instructions
1. Make sure Python is installed
2. Install required libraries:
3. Run the project

## How it works
1. Movies are stored with genre tags (action, scifi, drama, etc.)
2. Cosine similarity is calculated between all movies
3. Top 3 most similar movies are recommended

## Results
- Enter any movie name from the list
- System returns 3 similar movies based on genre
- Example: Input "Inception" → Output: Interstellar, The Dark Knight, Avengers
